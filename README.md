# DevChallenge

Code implemented on Dev Challenge assessment. The aim of this project is convert data from csv file to Json objects.

## Requirements

- Run npm install to install requeriments

### Libraries used:

- csv-parser
- google-libphonenumber

## How to run

```console
npm start
```
